##### 단계 1 - 팩토리얼 (10872)

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1)

N = int(input())

answer = factorial(N)

print(answer)
```

##### 단계 2 - 피보나치 수 5 (10870)

```python
def fibonacci(n):
    if n >= 2:
        return fibonacci(n-1) + fibonacci(n-2)
    else:
        return n

n = int(input())

answer = fibonacci(n)

print(answer)
```
